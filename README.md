# SérieSN Explorer

Application web React pour la découverte des séries sénégalaises.

##  Aperçu

> Catalogue · Détail · Favoris · Mode sombre/clair

##  Installation

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:5173](http://localhost:5173)

##  Structure
src/
├── components/   → Navbar,SerieCard   SearchBar, FilterBar, StarRating, Loader
├── context/      → FavorisContext, ThemeContext
├── data/         → series.json
├── pages/        → Accueil, Detail, Favoris
├── styles/       → index.css
└── App.jsx

##  Technologies

- React 18
- React Router DOM v6
- Vite
- CSS
